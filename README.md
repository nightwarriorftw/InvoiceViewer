## :zap: Usage
Just run it !!

### :nut_and_bolt: Development Environment

#### 1. Clone the Repository

```Bash
git clone https://github.com/nightwarriorftw/InvoiceViewer.git
cd InvoiceViewer
```

#### 2. Install dependencies
```Bash
yarn install
```

#### 3. Run the project:

```BASH
yarn start
```
